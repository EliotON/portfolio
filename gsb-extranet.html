GSB EXTRANET

GSB EXTRANET est une application moderne dédiée à la gestion des médecins pour le laboratoire GSB. Elle facilite le suivi des informations des médecins et des interactions avec eux.

Technologies utilisées :

  • PHP
  • Tailwind CSS
  • MySQL

Fonctionnalités principales :

  • Gestion des produits
  • Suivi des interactions
  • Base de données sécurisée
